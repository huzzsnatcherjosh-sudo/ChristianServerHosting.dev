Getting Started Guide

Everything you need to know to start hosting

Quick Start Guide

Step 1: Create Your Account

  1. Click "Get Started Free" on the homepage
  2. Enter your email address and choose a password
  3. Verify your email (check spam folder if needed)
  4. Log in to your dashboard

Step 2: Create Your First Server

  1. Click "Create New Server" in your dashboard
  2. Select your server type (Minecraft, Discord Bot, Web, etc.)
  3. Choose your server location
  4. Pick your plan (all plans are free!)
  5. Click "Deploy Server"

Your server will be ready in under 30 seconds!

Step 3: Configure Your Server

  1. Access your server control panel
  2. Customize settings through the easy-to-use interface
  3. Upload plugins, mods, or custom files
  4. Start your server and share the IP with friends

Control Panel Guide

Dashboard Overview

Your control panel provides everything you need to manage your server:

  • Console: Execute commands and view real-time logs
  • File Manager: Upload, edit, and organize files
  • Statistics: Monitor CPU, RAM, disk, and network usage
  • Backups: Create and restore backups with one click
  • Settings: Configure server parameters
  • Plugins/Mods: Browse and install from our library

File Manager

The built-in file manager lets you:

  • Upload files via drag-and-drop
  • Edit files with syntax highlighting
  • Create folders and organize structure
  • Download files and folders as ZIP
  • Set file permissions
  • Search for specific files

FTP/SFTP Access

For advanced users, we provide FTP and SFTP access:

  1. Navigate to "FTP Access" in your control panel
  2. Generate FTP credentials
  3. Use FileZilla, WinSCP, or any FTP client
  4. Connect using the provided credentials

Common Tasks

Installing Plugins (Minecraft)

  1. Stop your server
  2. Go to "Plugins" tab in control panel
  3. Search for your desired plugin
  4. Click "Install" and wait for completion
  5. Start your server

Creating Backups

  1. Navigate to "Backups" tab
  2. Click "Create Backup"
  3. Wait for backup to complete (usually under 1 minute)
  4. Download backup or restore it later

Changing Server Version

  1. Stop your server
  2. Go to "Settings" â†’ "Server Version"
  3. Select your desired version
  4. Click "Update" and wait for installation
  5. Start your server

Connecting a Custom Domain

  1. Purchase a domain from any registrar
  2. Add an A record pointing to your server IP
  3. In control panel, go to "Domains"
  4. Add your custom domain
  5. SSL certificate will be auto-generated

Troubleshooting

Server Won't Start

  • Check console logs for error messages
  • Ensure you have enough RAM allocated
  • Verify all plugins are compatible with your version
  • Try removing recently added plugins
  • Contact support if issue persists

Connection Issues

  • Verify server is running (green status)
  • Check you're using the correct IP and port
  • Ensure firewall isn't blocking connection
  • Try restarting your server
  • Check our status page for any outages

Performance Issues

  • Monitor resource usage in dashboard
  • Reduce view distance if using Minecraft
  • Remove unnecessary plugins
  • Optimize your world (remove excess entities)
  • Consider upgrading to a higher plan

API Documentation

For developers, we offer a comprehensive REST API:

Getting Started with API

  1. Generate an API key in your account settings
  2. Include the key in request headers: Authorization: Bearer YOUR_API_KEY
  3. Base URL: https://api.christanshosting.dev/v1/

Common Endpoints

  • GET /servers - List all your servers
  • POST /servers - Create a new server
  • GET /servers/{id} - Get server details
  • POST /servers/{id}/start - Start server
  • POST /servers/{id}/stop - Stop server
  • POST /servers/{id}/restart - Restart server
  • GET /servers/{id}/stats - Get server statistics
